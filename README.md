# Hi, I'm Thiago C Silva

### My Engineering Philosophy

I believe that true proficiency in technology doesn't come from memorizing syntax or mastering a specific tool, but from understanding the first principles that govern computing.

Whether I am building a backend in Java or scripting an automation in Bash, my approach is always to look "under the hood". This mindset allows me not just to fix bugs, but to solve problems at their root cause.

### Why Open Source Matters

My commitment to the Open Source ecosystem goes beyond using free software; it is about contribution and responsibility. I believe that contributing to "upstream" projects is the most effective way to improve the daily lives of developers everywhere.

When we fix a bug in a foundational tool, we aren't just solving a local problem, we are saving time for thousands of colleagues around the world.
